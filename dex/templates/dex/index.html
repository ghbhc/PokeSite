Thanks for visiting my Pokedex website! This is a project I did for fun and professional development using Django. The original 151 Pokemon are one of my favorite go-to data sets to work with because they have a lot of nostalgia value for me and they present some fun ways to think about and organize data.

To get the images and other information about the Pokemon for this webiste, I wrote a Python script to scrape data from pokemon.com. If you're interested in seeing any of templates, views, urls, etc... for this website, all of the source code can be found on my Github page here.

Please check out the different links in the top nav bar, or search for your favorite Gen 1 Pokemon in the search bar. You can browse to a particular Pokemon's detail page and click through all 151 of them.
